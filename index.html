Quem Somos

      Principal

      Este site trata-se de uma atividade proposta por nosso professor Angelo, na qual o cabeçalho possuí 3 âncoras e cada uma refere-se a uma categoria de mesmo nome na página principal e cada uma delas contém uma breve explicação sobre o que é referido nela.























      Produtos

      Nessa aplicação WEB oferecemos uma variedade de produtos, como conhecimento de HTML, CSS e Java Script, além de algumas noções de PHP, Java e C. Além disso, estamos dispostos a sempre dar dicas com relação as linguagens citadas anteriormente.
























      Quem Somos

      Nós somos um grupo de estudantes dos cursos de ADS e Cinência da Computação do Grupo Unis. Esse site trata-se de uma atividade da disciplina de Algorítmos e Progamação Web.Nossos integrantes são: Bernardo Clepf, Gabriel Braga, Gustavo Bueno, Jhonatan Oliveira e Kayky de Castro.
